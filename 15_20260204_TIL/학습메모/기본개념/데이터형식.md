## JSON(JavaScript Object Notation)
- 경량 데이터 교환 형식
- 사람이 읽고 쓰기 쉽고, 기계가 분석하고 생성하기 쉬움
- 주로 웹 애플리케이션에서 서버와 클라이언트 간 데이터 교환에 사용
- 예시:
 ```JSON
 {
   "name": "John Doe",
   "age": 30,
   "isStudent": false,
   "courses": ["Math", "Science", "History"],
   "address": {
     "street": "123 Main St",
     "city": "Anytown",
     "zip": "12345"
   }
 }
 ```

## XML(eXtensible Markup Language)
- 특정 목적에 맞게 데이터를 구조화하고 저장하기 위한 마크업 언어
 마크업이란? 데이터를 설명하는 태그를 추가하는 것
- 데이터를 계층 구조로 표현할 수 있으며, 사용자 정의 태그를 사용하여 다양한 형식의 데이터를 표현 가능
- 예시:
 ```XML
 <person>
  <name>John</name>
  <age>30</age>
  <city>New York</city>
</person>
```
