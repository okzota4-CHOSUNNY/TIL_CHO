## 생성자의 기본개념
1. 생성자(Constructor)란?
    생성자 = 생성하는자(사람, 도구)
    즉, 객체를 생성하는 특별한 함수.
2. 역할
 2-1. 객체가 생성될 때 자동으로 호출되어 객체 초기화 작업을 수행
 2-2. 객체의 필드(변수) 초기화
3. 특징
public class Car {
    private String color;

    public Car() {
        // 1. 클래스 이름과 똑같음(Car)
        // 2. 반환 타입이 없음(void도 안 붙임)
        // 3. 객체가 생성 될 때 자동으로 실행
        this.color = color;
        // 4. 초기값 설정
          // 4-1. 초기값은 없는 생성자 가능
          // 4-2. 초기값 설정하는 생성자 가능
          // 4-3. 고정값으로 초기화 가능
          // 4-4. 아예 생성자를 안만들어도 가능
    }
    Car myCar = new Car("빨강");
}
Car myCar = new Car();
 new Car()에서 생성자가 실행됨

## 생성자의 생성시기와 실행시기
생성자의 생성시기시가와 실행(호출)시기는 다름
1. 생성자가"생성될때"(=정의될 때)
public class Car{
    private String color;

    public Car(String color) {
        // 생성자 생성(정의)
        // 아직은 실행이 안됨, 정의만 해놓음
        this.color = color;
    }
}

2. 생성자가 "실행될 때"(=호출될 때)
Car myCar = new Car("빨강");
// new Car("빨강") ← 여기서 생성자가 "실행(호출)"됨
// this.color = color; ← 이 코드가 실행됨
