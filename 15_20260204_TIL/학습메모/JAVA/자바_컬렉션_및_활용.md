## List 및 ArrayList (가변 배열)
- **ArrayList**: 데이터 개수에 따라 크기가 자동으로 늘어나는 가변 배열.
- **주요 메서드**:
  - `add(item)`: 데이터 추가
  - `get(index)`: 데이터 조회
  - `set(index, item)`: 데이터 수정
  - `remove(index)`: 데이터 삭제
  - `size()`: 리스트 크기 확인

## new 키워드 사용 유무
- **new 불필요**: 리터럴 지원 타입 (String, Integer, Long 등 래퍼 클래스)
- **new 필요**: 사용자 정의 객체 (Board, User, Human 등 클래스)

## 람다식 (Lambda Expression)
- **정의**: 이름 없는 익명 함수. 코드를 간결하게 하고 함수형 프로그래밍을 지원함.
- **스트림(Stream)과 활용**: `list.stream().map(item -> item.getSome())` 처럼 데이터를 가공할 때 주로 사용.

## 정적 팩토리 메서드 (from)
- **from()**: Entity 객체를 DTO 객체로 변환할 때 관례적으로 사용하는 메서드 이름.
- **특징**: `public static`으로 선언하여 별도의 객체 생성 없이 클래스 이름으로 바로 호출 가능.
- **장점**: DTO 생성 로직을 한곳에 모아 관리할 수 있어 서비스 코드가 깔끔해짐.
