## 자바 핵심 키워드 정리

### 1. void (반환값 없음)
- **정의**: 메서드가 작업을 수행한 후 호출한 곳으로 아무런 값도 돌려주지 않을 때 사용.
- **예시**: `public void updateBoard(...)` -> 데이터를 수정만 하고 끝냄.
- **비교**: 반환값이 있는 경우 `public String getTitle()` 처럼 데이터 타입을 명시하고 `return` 키워드 사용.

### 2. final (최종적인)
- **변수에 사용**: 값을 한 번 정하면 바꿀 수 없음 (상수). 예: `public final double PI = 3.14;`
- **메서드에 사용**: 자식 클래스에서 오버라이딩(재정의)할 수 없음.
- **클래스에 사용**: 다른 클래스가 상속받을 수 없음. 예: `public final class String`.

### 3. 접근 제한자 (Access Modifier)
| 접근 제한자 | 클래스 내부 | 같은 패키지 | 상속 관계 | 전체 | 특징 |
| :--- | :---: | :---: | :---: | :---: | :--- |
| **public** | O | O | O | O | 어디서든 접근 가능 |
| **protected** | O | O | O | X | 상속 받은 자식까지 허용 |
| **default** | O | O | X | X | 같은 패키지 내에서만 허용 |
| **private** | O | X | X | X | 해당 클래스 내부에서만 허용 (캡슐화) |

### 4. static (정적)
- **정의**: 객체(인스턴스) 생성 없이 클래스 이름으로 바로 접근 가능한 변수나 메서드.
- **특징**: 프로그램 시작 시 메모리에 할당되며, 모든 인스턴스가 값을 공유함.
- **예시**: `Math.random()`, `BoardResponseDto.from()`

### 5. Getter와 Setter
- **Getter**: 클래스의 private 필드 값을 외부에서 **조회**할 때 사용 (`return` 있음).
- **Setter**: 클래스의 private 필드 값을 외부에서 **수정**할 때 사용 (`void`, `return` 없음).
- **Lombok 활용**: 클래스 상단에 `@Getter`, `@Setter`를 붙여 자동 생성 가능.
