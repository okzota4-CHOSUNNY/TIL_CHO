## 1. UUID (Universally Unique Identifier)
- 객체나 데이터의 고유한 식별자를 생성하는 데 사용.
- randomUUID() : 랜덤한 UUID 값을 생성하는 자바 내장 메서드.
- 형식: 32자리의 16진수 숫자를 하이픈으로 구분 (총 36자리).

## 2. 변수의 선언, 생성, 할당
- **선언(Declaration)**: `BoardRepository repo;` - 메모리 공간 확보 요청.
- **생성(Instantiation)**: `new BoardRepository();` - 실제 데이터를 메모리에 올림.
- **할당(Assignment)**: `repo = ...;` - 생성된 객체의 주소를 변수에 저장.
- **대입 연산자 (=)**: 오른쪽의 값을 왼쪽의 변수에 할당하는 역할.

## 3. 타입(Type) vs 이름(Name)
- **타입(Type)**: 데이터의 종류 (붕어빵 틀). 예: `BoardRepository`
- **이름(Name)**: 변수를 식별하기 위한 명칭 (실제 붕어빵). 예: `boardRepository`
- 자바 관례상 타입은 대문자로, 이름은 소문자로 시작함.

## 4. 메서드 선언 및 호출
- 메서드 뒤에는 항상 괄호 `()`가 붙어 있다.

## 5. 필드와 상태
- `private int enabled;`: 사용자의 활성화 상태 (1: 활성화, 0: 비활성화) 등을 나타내는 필드로 자주 쓰임.

## 6. 기본 생성자의 역할
- `public Users() { this.id = UUID.randomUUID().toString(); }`: 객체 생생 시 초기값(예: 고유 ID)을 자동으로 할당할 수 있음.
