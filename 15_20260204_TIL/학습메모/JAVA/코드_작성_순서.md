## 코드 입력순서
1. 1순위 라이브러리:
   로드도구들을 가져옵니다.
   ex: import java.util.List;, import java.util.ArrayList; 등

2. 2순위 상수 및 변수 설정:
   코드 전체에서 쓸 기준값들을 정합니다. 
   ex: final int MAX_SIZE = 100;, String title = "안녕하세요"; 등

6. 3순위 함수/클래스 정의:
   기능들을 미리 만들어 둡니다.
   ex: public class User { ... }, public void login() { ... } 등

7. 4순위 메인 실행 로직:
   실제로 프로그램이 돌아가는 부분입니다.
   ex: public static void main(String[] args) { ... } 등
