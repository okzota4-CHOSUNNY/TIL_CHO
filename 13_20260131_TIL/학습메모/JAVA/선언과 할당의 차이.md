```java
선언, 생성, 할당의 차이.
1. 선언(Declaration): 선언은 자바 컴파일러에 "이런 타입의 변수를 쓸거야" 메모리 공간을 확보해 달라고 요청하는 행위
BoardRepository boardRepository;
// 해석: BoardRepository 타입의 boardRepository 라는 이름을 가진 변수 선언

2. 할당(Initialization/Assignment): 선언된 변수에 생성된 객체의 주소를 집어넣는 단계. 우측에서 생성된 객체의 참조값(주소)을 좌측 변수에 할당
boardRepository = new BoardRepository();
// 해석: 생성된 BoardRepository 객체의 주소를 boardRepository 변수에 할당

3. 생성(Instantiation/Creation): 생성은 new키워드를 사용하여 실제 데이터를 메모리에 올리는 단계
new BoardRepository();
// 해석: BoardRepository 클래스의 실제 객체(인스턴스)를 메모리에 생성


예시코드
public class UserService {
    private final UserRepository userRepository;
    // 해석: 수정이 불가능한 UserRepository 타입의 userRepository 라는 이름의 변수 선언
    public UserService(UserRepository userRepository) {
    // 1. [public UserService] : "누구나 이 서비스를 호출해서 사용할 수 있다."
    // 2. [( ) 괄호] : "이 서비스가 작동하기 위해 반드시 받아와야 하는 '필수 조건'을 적는 입구다."
    // 3. [UserRepository userRepository] : "그 입구로 'UserRepository' 타입의 실제 데이터(객체 주소)를 배달받아 'userRepository'라는 임시 이름으로 쓰겠다."

    this.userRepository = userRepository;
    // 해석: "입구(매개변수)로 들어온 데이터를 내 진짜 칸(this)에 집어 넣는다."
    }
}
