public void updateBoard(String title, String content) {
    this.title = title;
    this.content = content;
1. updateBoard 메서드: 게시판 글의 제목과 내용을 수정하는 기능을 담당
2. title 매개변수: 수정할 새로운 제목
3. content 매개변수: 수정할 새로운 내용
// public: 접근제한자(모든곳에서 접근가능), void: 반환타입(아무것도 반환하지 않음), updateBoard: 메서드이름, (String title, String content): 매개변수선언
// 반환을 필요로 하지않는 모든곳에서 접근 가능한 메서드 updateBoard 선언
4. this.title: 현재 객체의 title 필드(기존 제목)
5. this.content: 현재 객체의 content 필드(기존 내용)
6. 사용예시: board.updateBoard("새 제목", "새 내용");
7. 전체 해석: 이 메서드는 게시판 글의 제목과 내용을 새로운 값으로 업데이트합니다.
8. 작동방식: 
   1. 이용자: 게시판에서 [수정] 버튼 클릭
   2. Controller: 어? 요청한 번호의 글 수정해달래 -> Service로 전달
   3. Service: 오케이, updateBoard("새 제목", "새 내용") 메서드 호출
   4. 결과: this.title과 this.content가 새로운 값으로 변경 
}
