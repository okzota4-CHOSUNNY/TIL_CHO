1. 첫번째 final 사용법
public final double PI = 3.14159;
해석: 이 PI 변수는 절대 변하지 않는 3.14159 라는 값입니다.
final: 이 값은 세상이 두쪽 나도 절대 변하지 않습니다.
PI = 3.14;
// 오류! final로 선언된 PI는 세상이 두쪽나지 않았기때문에 변경 불가
// 실제사용: 원주율, 중력가속도 등 변하지 않는 상수를 정의할 때 사용


2. 두번째 final 사용법
public final void login() {
해석: 누구나 접근 가능하지만 이 메서드는 절대 오버라이딩(재정의) 할수 없는 login 메서드
// 이 기능은 내가 만든게 무조건 맞으니, 자식들은 절대 이 기능을 변경하지 마세요!
// 실제사용: 보안이 중요한 기능(비밀번호 체크 등)을 자식클래스가 변경하지 못하도록 막음
// 안성재쉡의 모수 도토리국수 레시피: 맛이 변한다면 안성재가 아니거덩여!
}

3. 세번째 final 사용법
public final class String {
해석: 누구나 접근 가능하지만 상속 할수 없는 String 클래스
// 이 String 클래스는 절대 상속 하지마!
// 누군가 이 String 클래스를 상속받아 고쳐쓴다면 시스템이 엉망이됨.
// 실제사용: 핵심 라이브러리 클래스(java.lang.String 등)들이 상속되지 않도록 막음
// 정관수술을한 아들은 더이상 자식을 낳지 못함
}
